# EcoSort AI 🌱

AI-powered waste classification and climate impact tracking system.

## Features
- Waste image classification using AI
- CO₂ impact calculation
- Sustainability dashboard
- Recycling recommendations
- Climate education module

## Tech Stack
- Python
- Streamlit
- TensorFlow
- SQLite
- Plotly

## Run Project
pip install -r requirements.txt
streamlit run app.py
